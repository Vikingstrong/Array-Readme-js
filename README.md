# Массивы и методы для массивов

## Что такое массивы в Java Script ?

Массивы в JS это обьектный тип данных который в отличии от примитивов (string, number и тд...) может хранить в себе несколько значений. У массивов есть элементы и индексы. Элементы это само содержание массивов а с помощью индексов мы можем обращаться к элементам массива.

``` js
    let myArr = []; // Так создается пустой массив
    
                //0 1 2 <= это индексы элементов
    let myArr2 = [1,2,3]; // В этом же массиве у нас есть 3 элемента 1,2,3
```

Элементы массива так же можно менять по их индексу добавлять или изменять уже существующие элементы.

```js
    let arr = [1,2,3];
    arr[3] = 8;
    console.log(arr); // [1,2,3,8]
```
Или же можно изменять уже созданые элементы

```js
    let arr = [1,2,3];
    arr[2] = "Hello, World!";
    console.log(arr); // [1,2,'Hello, World!']
```

Так же если создать новый элемент в масиве на перёд оставив преведущие индексы пустыми они будут вывовидтся как пустые ячейки

```js
    let arr = [1,2,3];
    arr[6] = "hi";
    console.log(arr); // [ 1, 2, 3, <3 empty items>, 'hi' ]
```

# Методы массива
## Методы массива делятся на 3 основных типа:

### Метод изменяющий изначальный массив
Такие методы изменяют основной массив внося в него изменения.

Методы которые входят в эту категорию:

``` js
    shift();
    unshift();
    pop();
    push();
    splice();
```
Коротко как работают такие методы: 

``` js
    let arr = [1,2,3];
    arr.push(4);
    console.log(arr); // [1,2,3,4]
```

### Методы не изменяющий изначальный массив внося изменения в его копию

Такие методы работают совсем по другому. Когда вы приминяете такие методы основной массив не изменяется и сохраняется. Создается его копия в которую и вносятся изменения.

Методы которые входят в эту категорию:

``` js
    concat();
    slice();
    join();
    toString();
    toReversed();
```

Коротко как работают такие методы: 

``` js
    let arr1 = [1,2];
    let arr2 = [3];
    let totalarr = arr1.concat(arr2);

    console.log(totalarr); // [1,2,3];
```
Суть в том что используя метод мы сохраняем его в переменую, однако если мы это не сделаем метод применится но результат никуда не сохранится.

``` js
    let arr1 = [1,2];
    let arr2 = [3];
    arr1.concat(arr2);
    
    console.log(arr1) // [1,2] поскольку результать не сохранен никуда
```

### Методы возвращающие новое значение
Такие методы не изменяют массив а просто возвращают какое либо значение в ходе использования. 

Методы которые входят в эту категорию:

``` js
    includes();
    indexOf();
```

Коротко как работают такие методы: 

``` js
    let arr = [1,2,3,4,5];
    console.log(arr.indexOf(3)); // 2 поскольку индекс 3 это 2 
```

Такие методы работают очень просто. Метод выполняется и возвращает какое либо значения не меняя ничего не в масиве и не в его копии.

### Метод push();

Метод push() добавляет одно или несколько новых значений в конец массива

``` js
    let arr = [1,2,3];
    arr.push(5);
    console.log(arr); // [1,2,3,5];
```

### Метод pop();

Метод pop() удаляет последний элемент в массиве и вовзращает его.

``` js
    let arr = [1,2,3];
    arr.pop();
    console.log(arr); // [1,2]
```

### Метод unshift();

Метод unshift() добавляет один или несколько элементов в начало масива

``` js
    let arr = ["hi", 3, 5];
    arr.unshift("bye");
    console.log(arr); // ["bye", "hi", 3, 5]
```

### Метод shift();
Метод shift() работает как pop() только он удаляет элемент из начала массива и возвращает его

``` js
    let arr = [1,2,3];
    arr.shift();
    console.log(arr); // [2,3];
```

### Метод concat();

Метод concat обьеденяет 2 или несколько массивов в 1 единый массив

``` js
    let arr1 = [1,2,3];
    let arr2 = [4,5,6];
    let arr3 = arr1.concat(arr2);
    console.log(arr3); // [1,2,3,4,5,6]
```

### Метод slice();

Метод slice делает вырез массива имея 2 значения start end. Если не задать ему 2 значение он пойдет до конца начиная с стартового значения.

``` js
    let arr = [1,2,3,5];
    console.log(arr.slice(0,3)); // [1,2,3];
```

### Метод join();

Метод join превращяет массив в строку с заданым аргументом

``` js
    let arr = ["hi", "bye", "hello"];
    console.log(arr.join("+")); // "hi+bye+hello"
```

### Метод splice();

Метод splice похож на slice однако он может принимать больше 2 значений и имеет сильную разницу. у него 2 основных значения start, deleteCount. Старт откуда начинать deleteCount какой элемент (по индексу) удалять. Так же можно задать еще больше значений item1,item2,item3... Это на что будет заменятся наш удаленый
элемент

``` js
    let arr = [2,3];
    let arr2 = arr.splice(0,0,1); // начал с 0 индекса и на 0 индекс добавил элемент 1
    console.log(arr2); // [1,2,3]   
```

### Метод toReversed();

Метод toReversed разворачивает массив не меняя исходный вариант

``` js
    let arr = [1,2,3];
    let rev = arr.toReversed();
    console.log(rev); // [3,2,1]
```
